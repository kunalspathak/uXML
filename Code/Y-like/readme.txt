Y parser for demonstrating dynamic typed programming language. 
- No Variable declaration required.
- Performs type checking at run-time instead of compile-time
- Arrays, map/reduce, functions (including lambda)

Features :
- Passing values of different data-type to same function.
- Passing functions as argument to a function. 
- Passing array to a function.
- Passing array of function to a function.
- Map function.
- Map/reduce using lambda function.
